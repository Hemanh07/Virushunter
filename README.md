# Virus Hunter

  ## to run this project you need to install django package in your local system

  
  ### for linux and macos use:
  
    pip3 install django
              
### for windows use:

    pip install django  
## to run the project


### for linux and macos use:

    python3 manage.py runserver
              

### for windows use:

    python  manage.py runserver
